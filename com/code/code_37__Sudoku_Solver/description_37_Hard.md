<div class="question-description__3U1T" style="padding-top: 10px;"><div><p>Write a program to solve a Sudoku puzzle by filling the empty cells.</p>

<p>A&#160;sudoku solution must satisfy <strong>all of&#160;the following rules</strong>:</p>

<ol>
	<li>Each of the digits&#160;<code>1-9</code> must occur exactly&#160;once in each row.</li>
	<li>Each of the digits&#160;<code>1-9</code>&#160;must occur&#160;exactly once in each column.</li>
	<li>Each of the the digits&#160;<code>1-9</code> must occur exactly once in each of the 9 <code>3x3</code> sub-boxes of the grid.</li>
</ol>

<p>Empty cells are indicated by the character <code>'.'</code>.</p>
