<div class="question-description__3U1T" style="padding-top: 10px;"><div><p>

</p>

<p><b>Example 1:</b><br/>
</p><pre><b>Input:</b> [1,1,2,3,3,4,4,8,8]
<b>Output:</b> 2
</pre>
<p/>

<p><b>Example 2:</b><br/>
</p><pre><b>Input:</b> [3,3,7,7,10,11,11]
<b>Output:</b> 10
</pre>
<p/>

<p><b>Note:</b>
Your solution should run in O(log n) time and O(1) space.
