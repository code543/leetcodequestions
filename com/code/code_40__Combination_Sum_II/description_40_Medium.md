<div class="question-description__3U1T" style="padding-top: 10px;"><div><p>Given a collection of candidate numbers (<code>candidates</code>) and a target number (<code>target</code>), find all unique combinations in <code>candidates</code>&#160;where the candidate numbers sums to <code>target</code>.</p>

<p>Each number in <code>candidates</code>&#160;may only be used <strong>once</strong> in the combination.</p>

<p><strong>Note:</strong></p>

<ul>
	<li>All numbers (including <code>target</code>) will be positive integers.</li>
	<li>The solution set must not contain duplicate combinations.</li>
</ul>

<p><strong>Example 1:</strong></p>

<pre><strong>Input:</strong> candidates =&#160;<code>[10,1,2,7,6,1,5]</code>, target =&#160;<code>8</code>,
<strong>A solution set is:</strong>
[
  [1, 7],
  [1, 2, 5],
  [2, 6],
  [1, 1, 6]
]
</pre>

<p><strong>Example 2:</strong></p>

<pre><strong>Input:</strong> candidates =&#160;[2,5,2,1,2], target =&#160;5,
<strong>A solution set is:</strong>
[
&#160; [1,2,2],
&#160; [5]
]
</pre>
