<p>For example, with A = "abcd" and B = "cdabcdab".</p>

<p>Return 3, because by repeating A three times (&#8220;abcdabcdabcd&#8221;), B is a substring of it; and B is not a substring of A repeated two times ("abcdabcd").</p>

<p><b>Note:</b><br/>
The length of <code>A</code> and <code>B</code> will be between 1 and 10000.</p>
