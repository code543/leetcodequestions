<div class="question-description__3U1T" style="padding-top: 10px;"><div><p>Two elements of a binary search tree (BST) are swapped by mistake.</p>

<p>Recover the tree without changing its structure.</p>

<p><strong>Example 1:</strong></p>

<pre><strong>Input:</strong> [1,3,null,null,2]

&#160;  1
&#160; /
&#160;3
&#160; \
&#160;  2

<strong>Output:</strong> [3,1,null,null,2]

&#160;  3
&#160; /
&#160;1
&#160; \
&#160;  2
</pre>

<p><strong>Example 2:</strong></p>

<pre><strong>Input:</strong> [3,1,4,null,null,2]

  3
 / \
1   4
&#160;  /
&#160; 2

<strong>Output:</strong> [2,1,4,null,null,3]

  2
 / \
1   4
&#160;  /
 &#160;3
</pre>

<p><strong>Follow up:</strong></p>

<ul>
	<li>A solution using O(<em>n</em>) space is pretty straight forward.</li>
	<li>Could you devise a constant space solution?</li>
</ul>
