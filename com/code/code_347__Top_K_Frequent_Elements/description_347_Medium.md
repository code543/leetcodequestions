<div class="question-description__3U1T" style="padding-top: 10px;"><div><p>Given a non-empty array of integers, return the <b><i>k</i></b> most frequent elements.</p>

<pre><strong>Input: </strong>nums = <span id="example-input-1-1">[1,1,1,2,2,3]</span>, k = <span id="example-input-1-2">2</span>

</pre>

<div>

</div>
