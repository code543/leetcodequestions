##[440.K-th Smallest in Lexicographical Order](https://leetcode.com/problems/k-th-smallest-in-lexicographical-order/description/ "440.K-th Smallest in Lexicographical Order")
######Difficulty:Hard

<div class="question-description__3U1T" style="padding-top: 10px;"><div><p>Given integers <code>n</code> and <code>k</code>, find the lexicographically k-th smallest integer in the range from <code>1</code> to <code>n</code>.</p>

<p>Note: 1 &#8804; k &#8804; n &#8804; 10<sup>9</sup>.</p>

<p><b>Example:</b>
</p><pre><b>Input:</b>
n: 13   k: 2

<b>Output:</b>
10

<b>Explanation:</b>
The lexicographical order is [1, 10, 11, 12, 13, 2, 3, 4, 5, 6, 7, 8, 9], so the second smallest number is 10.
