<div class="question-description__3U1T" style="padding-top: 10px;"><div><p/><p>Given a <b>non-empty</b> integer array, find the minimum number of moves required to make all array elements equal, where a move is incrementing a selected element by 1 or decrementing a selected element by 1.</p>

<p>You may assume the array's length is at most 10,000.</p>

<p><b>Example:</b>
</p><pre><b>Input:</b>
[1,2,3]

<b>Output:</b>
2

<b>Explanation:</b>
Only two moves are needed (remember each move increments or decrements one element):

[1,2,3]  =&gt;  [2,2,3]  =&gt;  [2,2,2]
