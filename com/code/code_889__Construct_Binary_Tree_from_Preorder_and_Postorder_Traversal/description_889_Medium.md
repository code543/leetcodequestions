##[889.Construct Binary Tree from Preorder and Postorder Traversal](https://leetcode.com/problems/construct-binary-tree-from-preorder-and-postorder-traversal/description/ "889.Construct Binary Tree from Preorder and Postorder Traversal")

<div class="question-description__3U1T" style="padding-top: 10px;"><div><p>Return any binary tree that matches the given preorder and postorder traversals.</p>

<p>Values in the traversals&#160;<code>pre</code> and <code>post</code>&#160;are distinct&#160;positive integers.</p>

<p>&#160;</p>

<div>
<p><strong>Example 1:</strong></p>

<pre><strong>Input: </strong>pre = <span id="example-input-1-1">[1,2,4,5,3,6,7]</span>, post = <span id="example-input-1-2">[4,5,2,6,7,3,1]</span>
<strong>Output: </strong><span id="example-output-1">[1,2,3,4,5,6,7]</span>
</pre>

<p>&#160;</p>

<p><strong><span>Note:</span></strong></p>

<ul>
	<li><code>1 &lt;= pre.length == post.length &lt;= 30</code></li>
	<li><code>pre[]</code> and <code>post[]</code>&#160;are both permutations of <code>1, 2, ..., pre.length</code>.</li>
	<li>It is guaranteed an answer exists. If there exists multiple answers, you can return any of them.</li>
</ul>
