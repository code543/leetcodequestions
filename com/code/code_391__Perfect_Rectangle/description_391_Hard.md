##[391.Perfect Rectangle](https://leetcode.com/problems/perfect-rectangle/description/ "391.Perfect Rectangle")
######Difficulty:Hard
####Similiar Topics:

####Similiar Problems:

<div class="question-description__3U1T" style="padding-top: 10px;"><div><p>
Given N axis-aligned rectangles where N &gt; 0, determine if they all together form an exact cover of a rectangular region.
</p>

<p>
Each rectangle is represented as a bottom-left point and a top-right point. For example, a unit square is represented as [1,1,2,2]. (coordinate of bottom-left point is (1, 1) and top-right point is (2, 2)).
</p>

<div style="float:right"><img src="/static/images/problemset/rectangle_perfect.gif"/></div>
<p><b>Example 1:</b>
</p><pre>rectangles = [
  [1,1,3,3],
  [3,1,4,2],
  [3,2,4,4],
  [1,3,2,4],
  [2,3,3,4]
]

Return true. All 5 rectangles together form an exact cover of a rectangular region.
</pre>
<p/>

<div style="clear:both"/>

<div style="float:right"><img src="/static/images/problemset/rectangle_separated.gif"/></div>
<p><b>Example 2:</b>
</p><pre>rectangles = [
  [1,1,2,3],
  [1,3,2,4],
  [3,1,4,2],
  [3,2,4,4]
]

Return false. Because there is a gap between the two rectangular regions.
</pre>
<p/>

<div style="clear:both"/>

<div style="float:right"><img src="/static/images/problemset/rectangle_hole.gif"/></div>
<p><b>Example 3:</b>
</p><pre>rectangles = [
  [1,1,3,3],
  [3,1,4,2],
  [1,3,2,4],
  [3,2,4,4]
]

Return false. Because there is a gap in the top center.
</pre>
<p/>

<div style="clear:both"/>

<div style="float:right"><img src="/static/images/problemset/rectangle_intersect.gif"/></div>
<p><b>Example 4:</b>
</p><pre>rectangles = [
  [1,1,3,3],
  [3,1,4,2],
  [1,3,2,4],
  [2,2,4,4]
]

Return false. Because two of the rectangles overlap with each other.
</pre>
