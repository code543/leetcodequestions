<div class="question-description__3U1T" style="padding-top: 10px;"><div><p>Given a binary array, find the maximum length of a contiguous subarray with equal number of 0 and 1. </p>


<p><b>Example 1:</b><br/>
</p><pre><b>Input:</b> [0,1]
<b>Output:</b> 2
<b>Explanation:</b> [0, 1] is the longest contiguous subarray with equal number of 0 and 1.
</pre>
<p/>

<p><b>Example 2:</b><br/>
</p><pre><b>Input:</b> [0,1,0]
<b>Output:</b> 2
<b>Explanation:</b> [0, 1] (or [1, 0]) is a longest contiguous subarray with equal number of 0 and 1.
</pre>
<p/>

<p><b>Note:</b>
The length of the given binary array will not exceed 50,000.
