<div class="question-description__3U1T" style="padding-top: 10px;"><div><p>You are given an integer array <i>nums</i> and you have to return a new <i>counts</i> array. The <i>counts</i> array has the property where <code>counts[i]</code> is the number of smaller elements to the right of <code>nums[i]</code>.</p>

<p><b>Example:</b></p>

<pre><strong>Input:</strong> [5,2,6,1]
<strong>Output:</strong> <code>[2,1,1,0] 
<strong>Explanation:</strong></code>
To the right of 5 there are <b>2</b> smaller elements (2 and 1).
To the right of 2 there is only <b>1</b> smaller element (1).
To the right of 6 there is <b>1</b> smaller element (1).
To the right of 1 there is <b>0</b> smaller element.
