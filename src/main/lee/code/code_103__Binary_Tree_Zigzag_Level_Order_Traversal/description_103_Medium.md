<p>
For example:<br/>
Given binary tree <code>[3,9,20,null,null,15,7]</code>,<br/>
</p><pre>    3
   / \
  9  20
    /  \
   15   7
</pre>
<p/>
<p>
return its zigzag level order traversal as:<br/>
</p><pre>[
  [3],
  [20,9],
  [15,7]
]
</pre>
