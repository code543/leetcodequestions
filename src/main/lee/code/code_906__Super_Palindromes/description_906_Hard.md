##[906.Super Palindromes](https://leetcode.com/problems/super-palindromes/description/ "906.Super Palindromes")
######Difficulty:Hard
####Similiar Topics:
  [Math](https://leetcode.com//tag/math)

<div class="question-description__3U1T" style="padding-top: 10px;"><div><p>Let's say a positive integer is a&#160;<em>superpalindrome</em>&#160;if it is a palindrome, and it is also the square of a palindrome.</p>

<p>Now, given two positive&#160;integers <code>L</code> and <code>R</code> (represented as strings), return the number of superpalindromes in the inclusive range <code>[L, R]</code>.</p>

<p>&#160;</p>

<p><strong>Example 1:</strong></p>

<pre><strong>Input: </strong>L = <span id="example-input-1-1">"4"</span>, R = <span id="example-input-1-2">"1000"</span>
<strong>Output: </strong>4
<span><strong>Explanation</strong>: </span>4, 9, 121, and 484 are superpalindromes.
Note that 676 is not a superpalindrome: 26 * 26 = 676, but 26 is not a palindrome.</pre>

<p>&#160;</p>

<p><strong>Note:</strong></p>

<ol>
	<li><code>1 &lt;= len(L) &lt;= 18</code></li>
	<li><code>1 &lt;= len(R) &lt;= 18</code></li>
	<li><code>L</code> and <code>R</code> are strings representing integers in the range <code>[1, 10^18)</code>.</li>
	<li><code>int(L) &lt;= int(R)</code></li>
</ol>
